# Anitta Cosmos 2026

## Description
Ce projet présente les signes du zodiaque, les chakras, les orixás et des mandalas inspirés par Anitta en 2026. Il utilise **HTML** et **CSS** pour créer une interface interactive avec des effets de survol et des couleurs harmonieuses.

## Fonctionnalités
- Affichage des signes du zodiaque avec images et descriptions.
- Présentation des chakras avec couleurs et significations.
- Section spéciale pour les Orixás et la Mandala.
- Effets de hover sur les images pour interaction et dynamisme.
- Design responsive et moderne.

## Technologies utilisées
- HTML5
- CSS3
- Google Fonts (Roboto)
